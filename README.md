# Traffic Fine Monitoring System (Java)

A console-based Java mini project that simulates a traffic checking system.
The program collects details of riders, checks for underage driving, helmet usage, and license availability, and generates a final report of violations.

---

## Features

* Checks rider age eligibility
* Counts underage riders
* Detects riders without helmets
* Detects riders without licenses
* Generates a final summary report
* Processes multiple riders using loops

1. System asks details for each rider
2. Checks age eligibility
3. Records helmet and license violations
4. Displays final violation report
